搭建Blog的准备工作

  1. 安装有Git和Nodejs的Windows本地计算机（最好也有安装Vscode）
  2. Github账号
  3. 良好的网络环境和试错脾气

什么是Github Pages和Hexo

Gemini 2.0 flash：

GitHub Pages 是 GitHub 提供的静态网站托管服务。你可以将你的 HTML、CSS、JavaScript 等静态网页文件上传到 GitHub 仓库，然后 GitHub Pages 就会帮你把这些文件部署成一个可以通过网址访问的网站。

Hexo 是一个快速、简洁且高效的静态博客框架，它基于 Node.js。你可以使用 Markdown (或其他标记语言) 编写博客文章，然后 Hexo 会根据你选择的主题将这些文章生成静态网页。
参考网址：https://hexo.io/zh-cn/docs/
