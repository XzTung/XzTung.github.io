认证与连接

安装好Git后，可以通过SSH设置来解决私有仓库的连接与认证问题

生成SSH公钥与私钥

  • ssh-keygen是SSH生成公钥私钥的命令
  • -t ed25519指定了密钥类型为 Ed25519。ed25519表示一种SSH 密钥类型，具体来说，它是一种使用 EdDSA（椭圆曲线数字签名算法）协议的公钥算法，基于 Curve25519 椭圆曲线。
  • -C "your-email@example.com" 是密钥的注释字段，用于标记密钥的用途或所有者，通常会放置邮箱地址以便辨认。
    键入上述代码后回车，命令行提示输入文件名，可以指定自定义路径和文件名。

这将在本地生成一对 SSH 密钥：

  • 公钥通常保存在 ~/.ssh/id_ed25519.pub 中
